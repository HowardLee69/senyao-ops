# Senyao Ops

森耀內部運營系統。基於 [Atomic CRM](https://github.com/marmelab/atomic-crm) (MIT)。

## Phase 1: CRM

B2B 客戶管理 — 補習班、聯絡人、銷售 Pipeline、互動紀錄、任務提醒。

## Tech Stack

- React 19 + TypeScript + Vite
- react-admin 5 (shadcn-admin-kit)
- Supabase (PostgreSQL + Auth)
- Tailwind CSS + shadcn/ui

## Development

```bash
npm install
npm run dev
```

## Deployment

- Frontend: Vercel
- Backend: Supabase Cloud
